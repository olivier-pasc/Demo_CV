# Technology Stack

This document outlines the core technologies used in the CV Matcher AI project.

## Backend

*   **Primary Language:** Python
*   **Web Framework:** FastAPI (for building robust and high-performance APIs)
*   **Database:** Google Cloud Firestore (NoSQL document database for flexible data storage)
*   **File Storage:** Google Cloud Storage (for efficient and scalable storage of CVs)
*   **Artificial Intelligence/Machine Learning:** Google Vertex AI (Gemini 1.5 Pro) (for AI-powered CV analysis and candidate matching)
*   **PDF Processing:** pypdf (for extracting text and data from PDF resumes)

## Frontend

*   **Primary Language & Framework:** TypeScript with React 18 (for building dynamic and type-safe user interfaces)
*   **Build Tool:** Vite (for a fast and optimized development experience)
*   **Styling:** TailwindCSS (utility-first CSS framework for rapid and consistent UI development)
*   **Animation Library:** Framer Motion (for creating smooth and engaging UI animations)
*   **Data Fetching & Caching:** TanStack Query (for efficient management of server state)
*   **HTTP Client:** Axios (for making HTTP requests to the backend API)

## Deployment

*   **Platform:** Google Cloud Run (fully managed compute platform for deploying containerized applications)
